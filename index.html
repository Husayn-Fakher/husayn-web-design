RemindME:
Your Location-Based Task Manager

RemindME is a mobile app that helps users manage tasks with notifications based on location.
The app ensures users never forget important errands, such as buying groceries when near a store.
This project demonstrates best practices in software development, including Clean Architecture,
the MVVM design pattern, Asynchronous programming, Dependency Injection (DI), and CI/CD pipelines.
The GitHub repository provides access to the source code, along with the APK file, allowing users
to download and use the app.

Android App
Temperature probe

e-t°, ma sonde de température, is an Android application developed for real-time temperature monitoring.
Designed to work with proprietary sensors, the app enables users to configure their devices, view current temperature readings, and track temperature trends over time using dynamic graphs.
The application also features customizable alerts to notify users when the temperature exceeds or drops below preset thresholds, ensuring prompt action.
This project was developed during my tenure at EDF, where I served as the tech lead for the Android app.
I led the technical implementation, including sensor integration, user interface design, and performance optimization.
